一、Hello world

使用 props 跟 state 將參數帶入 component

  1. 使用 React、React-dom
  2. 安裝 React-MDL （好方便來著，大推）
  3. render 時，使用 <Hello date={new Date()} />，在 component 內部為 this.props.date
  4. component 內部使用 getInitialState 來取得元件初始化時的狀態，參數會在 this.state 下。
  5. input 中 value={{ this.state.name }} 必須搭配 onChange={this.handleChange} 等事件才能連動

二、事件綁定

  1. 使用 CommentBox import CommentItem 與 CommentForm。
  2. 參照 React 範例，在 CommentItem import Comment 做 markdown 轉換。
  3. Root：<CommentItem remove={this.remove}></CommentItem>
    Item：<a onClick={this.delete} ><Icon name="close" /></a>
    透過 onClick={this.delete}，將 item 或 index 傳送到 this.prop.remove() 的上層事件做處理
  4. 跟 vue.js 不同，必須將資料從 state 取出，splice 或 concat 後，再用 setState 設定回去。
